一、方法简介
----------------------------------------------------------------------
K-means:
  - 经典的划分聚类算法
  - 需要预设簇数K
  - 假设球形簇，对中心点进行迭代优化
  - 优点: 简单、快速、可解释性强
  - 缺点: 需要预设K、假设球形、对初始化敏感

HDBSCAN:
  - 基于密度的层次聚类算法
  - 自动确定簇数
  - 能识别任意形状的簇和噪声点
  - 优点: 自动簇数、识别噪声、任意形状
  - 缺点: 参数较多、计算较慢、解释性中等

五、方法选择建议
----------------------------------------------------------------------
选择K-means如果:
  1. 已知目标簇数（如本项目的512簇）
  2. 需要高度可解释性（簇中心明确）
  3. 计算资源有限（速度更快）
  4. 数据簇相对均匀且接近球形

选择HDBSCAN如果:
  1. 探索性分析，不确定簇数
  2. 数据包含噪声点
  3. 簇形状复杂、密度不均
  4. 需要层次化的簇结构信息
