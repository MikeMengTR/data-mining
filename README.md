# 数据挖掘课程设计 - 音乐编码聚类分析

## 项目概述
本项目针对音乐编码数据进行聚类分析，探索高维音乐特征的内在结构。

## 快速开始

### 环境配置
```bash
pip install -r requirements.txt
```

### 一键运行
```bash
bash run.sh
```

## 项目结构
```
├── data_preprocessing/          # 数据预处理
├── baseline_methods/            # 基础聚类方法（K-means等）
├── advanced_methods/            # 进阶聚类方法（HDBSCAN等）
├── reports_latex/               # LaTeX报告
├── DATASET.md                   # 数据集说明
└── requirements.txt             # 依赖包
```
